# 汉字笔画练习

> 在教小朋友写字的过程中，被吐槽写得丑，于是自己vibe了这么个小工具。

一个基于浏览器的汉字笔画学习工具，支持笔画演示与书写练习两种模式，适合汉字学习者练习笔顺。

![Kapture](./assets/Kapture.gif)

## 直接访问

体验地址：<https://www.sppan24.cn/hanzi-stroke.html>

## 功能特性

- **演示模式**：逐笔动画演示汉字书写顺序，支持单步前进、重复当前笔画、一键自动书写
- **练习模式**：在田字格中手写汉字，实时判断笔画正确性，错误超过阈值后自动显示提示
- **多字支持**：一次最多输入 10 个汉字，自动排列田字格，完成后自动跳转下一个字
- **成绩统计**：练习模式下实时统计正确笔画数与错误次数
- **自适应布局**：根据屏幕宽度自动调整田字格尺寸，兼容手机、平板和桌面端
- **田字格辅助线**：带十字与对角辅助线的田字格，帮助规范字形

## 使用方法

1. 在输入框中输入想要练习的汉字（最多 10 个）
2. 点击「加载」按钮（或按 Enter 键）
3. 选择模式：
   - **📖 演示**：点击「下一笔」逐步查看笔顺，或点击「自动写」观看完整动画
   - **✍️ 自己写**：在田字格中按正确笔顺手写，系统自动判断对错

> 项目为纯前端单页应用，无需后端，无需构建工具，直接打开 `index.html` 即可使用。

## 快速开始

```bash
# 克隆仓库
git clone <repo-url>
cd hanzi-stroke

# 直接用浏览器打开（推荐使用本地 HTTP 服务以避免跨域问题）
npx serve .
# 或
python3 -m http.server 8080
```

然后在浏览器中访问 `http://localhost:8080`。
